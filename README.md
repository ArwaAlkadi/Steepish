#  Steepish

Steepish is an iOS fitness application that turns daily steps into competitive fitness challenges with real-time tracking and gamified motivation.

## Features

- Solo and group step challenges
- Custom step goals and challenge duration
- Automatic step tracking using HealthKit
- Real-time progress tracking
- Competitive leaderboards
- Home Screen widget with a dynamic character reflecting activity
- Clean and minimal UI


## Tech Stack

- Swift
- SwiftUI
- HealthKit
- WidgetKit
- GameKit
- Firebase
- MVVM Architecture
- Xcode
